## Archivos necesarios para mustlisten (app standalone)

### Copiar tal cual:
app.py

### Base de datos (NO está en git — copiar manualmente):
db/must_hear_rym_new.db   →  must_hear_rym_new.db  (o la ruta que prefieras)

### Crear nuevo (solo Flask + gunicorn):
requirements.txt:
  flask>=3.0
  gunicorn

### Variables de entorno (crear .env o exportar):
  LASTFM_API_KEY=<tu_key>

### NO necesarios:
- Todo db/*.py, html_*.py, tools/, docs/
- La base de datos principal lastfm_cache_rym_new_normalized.db
- html_must_hear.py (es el generador estático, no el servidor)
- db/.env (contiene proxies, no relevante para la web app)

## Para correr localmente:
  pip install flask
  python app.py --db must_hear_rym_new.db --lastfm-api-key TU_KEY

## Para Fly.io:
  flyctl launch          # genera fly.toml
  flyctl volumes create data_vol --size 1    # volumen persistente para la DB
  # Ajustar fly.toml para montar el volumen en /data
  # Cambiar --db a /data/must_hear_rym_new.db
  flyctl deploy
